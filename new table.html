HTML ELEMENTS Description
<!DOCTYPE> Defines the document type and version of HTML being used
<html> Defines the root element of the HTML document
<head> Contains metadata about the HTML document
<title> Defines the title of the HTML document
<body> Defines the main content of the HTML document
<h1> to <h6> Defines headings of different levels
<p> Defines a paragraph
<br/> Inserts a single line brake
<hr/> Defines a thematic brake (horizontal rule)
FORMATTING
ELEMENTS 		Description
<b> Defines bold text
<strong> Defines important text (typically bold)
<i> Defines italic text
<em> Defines emphasized text (typically italic)
<mark> Defines marked/highlighted text
<ins> Defines inserted text (typically underlined)
<del> Defines deleted text (typically strikethrough)
<sup> Defines superscript text
<sub> Defines subscript text
